League Notes

Points system:
Win = 2 points
No Result = 1 point each
Loss = 0 points

NRR is calculated from match score entries:
(Runs scored / overs faced) - (Runs conceded / overs bowled)

The uploaded scoreboard screenshot is stored in your browser as match proof, so use the same device/browser for hosting this league.
