# League Samples

This folder contains quick test assets for [ipl-league-host.html](/Users/harshavaradhi/Documents/Claude/Artifacts/ipl-auction-app/ipl-league-host.html).

Files:
- `sample-ipl-teams-and-squads.md`: Copy-ready 4 IPL teams with squads
- `sample-league-export.json`: Example export structure matching the league app
- `result-screenshot-01-mi-vs-csk.svg`: Sample scoreboard image
- `result-screenshot-02-rcb-vs-kkr.svg`: Sample scoreboard image
- `result-screenshot-03-csk-vs-rcb.svg`: Sample scoreboard image

Quick test flow:
1. Open the league host app.
2. Copy team names and squads from `sample-ipl-teams-and-squads.md`.
3. Generate fixtures.
4. Open any fixture and enter the score shown in one of the sample screenshots.
5. Upload the matching `.svg` file as the scoreboard screenshot.

Note:
- The screenshots are original mock test images in a cricket video game style. They are not official game UI captures.
